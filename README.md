Getting Started
---------------

1.Get the latest snapshot:
git clone --depth=1 https://Fidenz-deshanr@bitbucket.org/Fidenz-deshanr/io2018_portal_node.git

2.Change directory:
cd myproject

3.Install NPM dependencies:
npm install

4.Then simply start app:
npm start
